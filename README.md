# 写真一括リネームツール（RENAME_INDEX.csv 対応）

GitHub Pages 上で動作する、ブラウザ完結型の写真一括リネームツールです。

- `RENAME_INDEX.csv` と JPG/PNG 画像ファイルを  
  **ブラウザにドラッグ＆ドロップ** するだけで、
- CSV に記載されたコードにしたがって
- 画像ファイル名を一括でリネームし、
- 新しいファイル名で ZIP をダウンロードできます。

処理はすべてブラウザ内（ローカル）で完結し、サーバーに画像は送信されません。

---

## 想定ユースケース

- 14条地図作成などで、  
  **撮影番号（0020.jpg）を「BR0020.jpg」のようなコードに変換したい**
- RENAME_INDEX.csv（A01 レコード）で撮影写真とコードの対応が定義されている

例：

- `0020.jpg` → `BR0020.jpg`
- `0021.jpg` → `BM0021.jpg`
- `0022.jpg` → `BL0022.jpg`
- `0023.jpg` → `BJ0023.jpg`

---

## 対応している CSV フォーマット

`RENAME_INDEX.csv` のうち、**`A01` 行の 3列目のコード** を使用します。

```csv
A01,2295,BR0020,-26450.296,-4324.762,,
A01,2296,BM0021,-26452.123,-4325.111,,
A01,2297,BL0022,-26453.456,-4326.222,,
A01,2298,BJ0023,-26454.789,-4327.333,,
CSVのインデックスに沿って画像ファイルをリネームします
